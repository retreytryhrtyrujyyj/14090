# springboot基于线性回归的音乐推荐系统的设计与实现(论文+PPT)

## 下载地址
https://bysj.bsitem.com/#/home/goodDetail?goodId=14090

## 技术介绍
Java语言 + SpringBoot框架（后端）+ Vue前端框架（Web）+ Mysql(数据库)

## 开发工具
IDEA或者MyEclipse + Navicat(mysql数据库可视化工具) + Tomcat

## 项目视频演示

## 项目资料包含
1. 前后端源代码
2. 开题报告（任务书）+ 配套论文（一万两千字以上）+ 答辩PPT
3. 远程控屏调试（向日葵或者toDesk控屏）
4. 一键启动项目（双击运行演示，无需搭建复杂环境）
5. 项目讲解视频教程
